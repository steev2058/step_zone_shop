# STEPZONE Shop

Minimal, fast sneaker store built with **Next.js + TailwindCSS**.

## Run

```bash
npm install
npm run dev
```

## Stack
- Next.js 14 (App Router)
- TailwindCSS
- Static product data for speed

## Notes
- Replace `/public/stepzone-logo.svg` with your final STEPZONE logo asset.
- All images are local and lazy-loaded by default.
